## Quick Reference

- **Language**: TypeScript with ES modules
- **Framework**: Serverless AWS Lambda
- **Database**: DynamoDB + Clickhouse
- **Validation**: Zod schemas
- **Middleware**: Middy
- **Authentication**: AWS Cognito
- **Encryption**: AWS KMS
